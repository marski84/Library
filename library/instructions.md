# Library

## Objectives

* Create an implementation for a simple library with booking system
* **Features to implement**:
    * Renting a book for a specific user
    * Return of a book - if is in deadline penalty is 0 if not every extra day is a one penalty
      point, when user reach 10 penalty points renting a book is unavailable for a month, after the
      month penalty is reset
    * Adding and removing books from the library
    * Adding and removing a user from the booking list
* Write tests for your implementation
